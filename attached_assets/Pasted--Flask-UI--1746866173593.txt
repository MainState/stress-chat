# 내 Flask 채팅 애플리케이션의 UI를 개선하고 싶습니다.
# 이번에는 채팅 메시지 입력창과 주요 버튼들(전송 버튼, 다시 시작하기 버튼)의 디자인을 더 세련되고 사용자 친화적으로 만들고 싶습니다.
# `static/style.css` 파일을 수정해주세요. (이 파일은 `templates/index.html`에 이미 링크되어 있다고 가정합니다.)

# === HTML 요소 가정 ===
# - 채팅 메시지 입력창: `<input type="text" id="user-input">`
# - 전송 버튼: `<button id="send-button">전송</button>`
# - 다시 시작하기 버튼 (결과 화면에 있음): `<button id="try-again-button">처음부터 다시 시작하기</button>`
# - 입력창과 전송 버튼을 감싸는 `div`가 있다면, 그 `div`에 `id="input-area"`를 부여했다고 가정하고 함께 스타일링해도 좋습니다. 없다면 AI가 필요에 따라 제안해줘도 됩니다.

# === 1. 메시지 입력창 (`#user-input`) 스타일 ===
#    a. `width`: 부모 요소 너비에 맞춰 유연하게 조절되도록 설정 (예: `flex-grow: 1;` 만약 부모가 flex라면, 또는 `width: calc(100% - (전송버튼너비) - (약간의간격));`). 또는 부모 요소에 `display: flex;`를 주고 입력창에는 `flex-grow: 1;`을, 버튼에는 고정 너비를 주는 방식을 제안해줘도 좋아.
#    b. `padding`: `10px 12px;` 정도로 적절한 안쪽 여백을 주세요.
#    c. `border`: 너무 두껍지 않은 실선 테두리 (예: `1px solid #ccc;`) 또는 아예 없애고 배경색으로 구분.
#    d. `border-radius`: `5px;` 정도로 부드러운 모서리를 주세요.
#    e. `font-size`: `1rem;` (또는 `16px`) 정도로 설정해주세요.
#    f. 포커스 시 효과: 사용자가 입력창을 클릭(포커스)했을 때, 테두리 색상이 살짝 변하거나(예: `#007bff`), 그림자 효과가 나타나도록 (`:focus` 가상 클래스 사용) 스타일을 추가해주세요.

# === 2. 전송 버튼 (`#send-button`) 스타일 ===
#    (이전에 버튼 기본 스타일을 적용했다면, 여기서는 전송 버튼에 특화된 아이콘 추가나 약간의 변형을 고려합니다.)
#    a. 이전에 정의한 버튼 기본 스타일을 따르되, 만약 가능하다면 버튼 텍스트 "전송" 대신 또는 함께 **종이비행기 모양의 아이콘(SVG 또는 폰트 아이콘)**을 사용하는 것을 제안해줘. (아이콘 사용이 복잡하다면, 텍스트 버튼 디자인을 더 다듬는 것으로 대체)
#    b. 입력창과의 간격: 입력창 오른쪽에 배치될 경우, 적절한 `margin-left`를 주세요.
#    c. 만약 `input-area` div를 flex로 구성한다면, 버튼의 높이가 입력창과 잘 맞도록 `align-items: center;` 등을 활용.

# === 3. "다시 시작하기" 버튼 (`#try-again-button` - 결과 화면 내) 스타일 ===
#    a. 이 버튼은 중요한 행동을 유도하므로, 기본 버튼 스타일을 따르되 조금 더 눈에 띄도록 할 수 있습니다. (예: 다른 배경색 - 녹색 계열 `#28a745`, 또는 테두리 스타일 강조)
#    b. 결과 화면 레이아웃에 맞춰 적절한 `margin-top` 등을 추가해주세요.

# === 4. (선택 사항) 입력 영역 전체 (`#input-area` - 입력창과 전송 버튼을 감싸는 div) 스타일 ===
#    - 만약 입력창과 전송 버튼을 `div id="input-area"`로 감쌌다면, 이 `div`에 `display: flex;` 와 `align-items: center;` 를 적용하여 두 요소가 한 줄에 보기 좋게 정렬되도록 해주세요.
#    - `margin-top: 15px;` 정도로 채팅창과의 간격을 주세요.

# 위 지시사항을 바탕으로 `static/style.css` 파일에 추가/수정할 CSS 코드를 작성해주고,
# 혹시 HTML 구조 변경이 필요하다면 (`input-area` div 추가 등) `templates/index.html` 파일의 어느 부분을 어떻게 수정해야 하는지도 함께 알려주면 좋겠습니다.
